# Nicla Vision

## Sensors/Inputs onboard
- Câmara
- Time-of-flight long distance ranging sensor
- Microfone omnidireccional
- IMU de 6-eixos (*inertial measurment unit*) (3 eixos acelerómetro + 3
  eixos giroscópio)

## Comunicação
- USB
- Bluetooth LE
- Wifi
